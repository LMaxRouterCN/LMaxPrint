# LMaxPrint - 轻量级图像编辑打印工具

## 1. 项目总结
开发一款轻量级桌面应用程序，主要用于图像的快速导入、基础编辑（旋转、缩放、镜像、平移）以及直接对接系统打印机进行打印或保存。

## 2. 具体目标
- **极简UI**：无圆角设计，符合用户偏好。左上角放置核心操作按钮（导入、打印、保存）。
- **图像变换**：支持鼠标拖拽平移、滚轮缩放、按钮/快捷键旋转和镜像。
- **打印集成**：自动获取系统默认打印机的纸张尺寸、DPI和边距参数；支持手动修改打印参数。
- **事件驱动**：避免硬编码延时，所有UI更新和打印任务均基于事件回调，绝不卡死主线程。

## 3. 技术栈选择
- **语言/框架**：C# / .NET 8 WinForms
- **理由**：WinForms 调用 `System.Drawing.Printing` 和 `System.Drawing.Drawing2D` (Matrix) 最为原生和轻量，无需引入沉重的第三方UI库，打包后体积极小，完美契合"轻量"需求。
